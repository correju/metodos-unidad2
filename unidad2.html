Problema 1

La siguiente figura sugiere como estimar el valor de \(\pi\) con una simulacin. En la figura, un circuito con un rea igual a \(\pi/4\), est inscrito en un cuadrado cuya rea es igual a 1. Se elige de forma aleatoria \(n\) puntos dentro del cuadrado. La probabilidad de que un punto est dentro del crculo es igual a la fraccin del rea del cuadrado que abarca a este, la cual es \(\pi/4\). Por tanto, se puede estimar el valor de \(\pi/4\) al contar el nmero de puntos dentro del crculo, para obtener la estimacin de \(\pi/4\). De este ltimo resultado se encontrar una aproximacin para el valor de \(\pi\).

Pasos sugeridos:

Genere n coordenadas \(x: X1, . . . , Xn\). Utilice la distribucin uniforme con valor mnimo de 0 y valor mximo de 1. La distribucin uniforme genera variables aleatorias que tienen la misma probabilidad de venir de cualquier parte del intervalo (0,1).

Problema 2

La simulacin ayuda a entender y validad las propiedades de los estimadores estadsticos como son. insesgadez, eficiencia y la consistencia principalmente. El siguiente problema permite evidenciar las principales caractersticas de un grupo de estimadores propuestos para la estimacin de un parmetro asociado a un modelo de probabilidad.

Sean \(X1, X2, X3 \,y\, X4\), una muestra aleatoria de tamao \(n=4\) cuya poblacin la conforma una distribucin exponencial con parmetro \(\theta\) desconocido. Determine las caractersticas de cada uno de los siguientes estimadores propuestos:

n <- 4
m = 5000
lam <- 0.5

rep = rexp(m*n,rate = lam)
df = data.frame(matrix(rep,nrow = m, ncol = n, byrow = TRUE))

estimador1 <- function(m) { ((m[1] + m[2]) / 6) + ((m[3] + m[4]) / 3) }
estimador2 <- function(m) { (m[1] + 2*m[2] + 3*m[3] + 4*m[4]) / 5 }
estimador3 <- function(m) { mean(m) }
estimador4 <- function(m) { (min(m) + max(m)) / 2 }

estimarMuestra <- function(data, n_muestra, lam) {
  teta <- 1 / lam
  muestra <- sample_n(df, n_muestra)
  t1 <- apply(df, 1, estimador1)
  t2 <- apply(df, 1, estimador2)
  t3 <- apply(df, 1, estimador3)
  t4 <- apply(df, 1, estimador4)
  media <- apply(data.frame(t1, t2, t3, t4), 2, mean)
  varianza <- apply(data.frame(t1, t2, t3, t4), 2, var)
  sesgo <- media - teta
  boxplot(t1, t2 ,t3, t4, col = c("#377eb8", "#ff7f00", "#4daf4a", "#f781bf"), main = paste("n = ", n_muestra))
  abline(h=teta, col="red", lwd=2)
  return (data.frame(media = media, varianza = varianza, n=n_muestra, sesgo=sesgo))
}

res20 <- estimarMuestra(df, 20, lam)

res50 <- estimarMuestra(df, 50, lam)

res100 <- estimarMuestra(df, 100, lam)

res1000 <- estimarMuestra(df, 1000, lam)
